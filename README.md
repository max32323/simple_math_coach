Простой и удобный в использовании тренажёр, который помогает развить вычислительные способности.
Основная суть приложения - решение примеров на счёт. 
Всего есть 3 уровня: легкий, средний, сложный.  

Приложение предназначено для людей, которые испытывают сложности со счетом(в основном для детей начальных классов).
Надеюсь он вам очень понравиться.
